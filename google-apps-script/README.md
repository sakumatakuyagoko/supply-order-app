# Google Apps Script セットアップガイド

ReactアプリとGoogle Sheetsを連携させるための手順です。以下のステップに従って設定を行ってください。

## 1. Google Sheet の準備
1. 新しい Google Sheet を作成してください。
2. 最初のシート（タブ）の名前を `Products` に変更してください。
3. `Products` シートの1行目に、以下のヘッダーを追加してください（順不同可）:
   `id`, `name`, `category`, `price`, `unit`, `stockStatus`, `image`
4. 2行目以降に、テスト用の商品データをいくつか入力してください。
5. 2つ目のシートを作成し、名前を `Orders` に変更してください。
6. `Orders` シートの1行目に、以下のヘッダーを追加してください:
   `orderId`, `timestamp`, `items_json`, `totalAmount`, `status`

## 2. スクリプトのデプロイ
1. Google Sheet のメニューから `拡張機能` > `Apps Script` をクリックします。
2. このフォルダにある `Code.js` の内容をコピーして、スクリプトエディタに貼り付けてください（元のコードは削除してOKです）。
3. 右上の `デプロイ` ボタン > `新しいデプロイ` をクリックします。
4. 左上の歯車アイコン > `ウェブアプリ` を選択します。
5. 以下のように設定します:
   - **説明**: Supply Order API（任意）
   - **次のユーザーとして実行**: 自分（メールアドレス）
   - **アクセスできるユーザー**: **全員** (アプリからログインなしでアクセスするために必須です)
6. `デプロイ` をクリックします。
   - ※初回はアクセス承認の画面が出ることがあります。「アクセスを承認」→ アカウント選択 → （警告が出たら）詳細表示 → 「安全ではないページに移動」の手順で許可してください。
7. 表示された **「ウェブアプリURL」** をコピーしてください。

## 3. アプリとの連携
1. コピーしたウェブアプリURLを、AIアシスタント（私）に教えてください。アプリの設定ファイルに反映させます。
